CSS Grid Layout, is a two-dimensional grid-based layout system with rows and columns that simultaniously workes at a time. It allow us to align elements into columns and rows using its layout. On the other hand CSS Flexbox offers a one-dimensional layout. It is helpful in allocating and aligning the space among items in a container. It works with all kinds of display devices and screen sizes. Major Uniqueness between Flexbox and Grids is that the former works on content while the latter is based on the layout.

Bootstrap is an open-source framework that is used to design responsive websites better, faster, and easier.
